Covid19 Data Visualization

Infection and death information

Tableau Public Viz

Summary:

This is the visualization of my exploration into Covid infections and deaths data. It consists of a table, a bar graph, an interactive heat map, and a line graph with predictions. I followed the Alex the Analyst YouTube video to complete this project. In the future I plan on exploring the data further to view how different comorbidities affected a patient's outcome.

Data Source:

Our World in Data website (August 22, 2023).

Skills & Programs Used:

  • Tableau
  • Data Visualization

Process

Gather Data

Get data from data tables in BigQuery using SQL and export as MS Excel spreadsheets.

Create Tableau Sheets

Using the saved Excel tables, create graphs in Tableau sheets.

Create Tableau Dashboard

Place the graphs created on each sheet onto a dashboard.

Publish Dashboard

Publish the finished dashboard to Tableau Public.

Results

This is the resulting dashboard created on Tableau Public that displays Covid19 death and infection information. On the left side it shows the total cases, total deaths, and deathpercentage as a table and a bar graph of the total death count for each continent. On the right side, an interactive map showing the highest percent of a country's population infected. At the bottom a line graph shows the percent of the population of four countries infected by month from November 2019 to July 2023. The line graph also includes predictions (shaded area) for the next year.

As seen on the varous charts, the areas with the highest infection and death rate from Covid19 were Europe, Asia, and North America. At the end of August 2023, the total number of Covid19 cases was 769,789,653 and the death rate was 0.90%.

Step 1: Gather Data

Get data from data tables (in BigQuery SQL) and save results as MS Excel file.

  1. Get total cases, deaths, and calculated death percentage from dataset.
  2. Remove locations not in previous query (European Union, World, High income, Upper middle income, Lower middle income, Low income)
  3. Get total deaths per continent from dataset.
  4. Calculate the percent of population infected from the highest infection count per location.
  5. Get the corresponding dates for the calculated percent of population infected per location.
  6. Change nulls in Excel columns to 0.
×

Step 2: Create Tableau Sheets

Import each table from Step 1 into seperate Tableau sheets.

  1. Sheet 1: Global numbers (Table)
  2. Sheet 2: Current global death count (Bar Graph)
  3. Sheet 3: Total Infection Percentage by country (Map)
  4. Sheet 4: Highlight select countries (Line Graph with predictions)
×

Step 3: Create Tableau Dashboard

Place Tableau sheets on dashboard.

  1. Add a desktop layout and change fit from "Default" to "Fit All".
  2. Place the Global Numbers table in the top left corner.
  3. Place the Current Global Death Count bar graph in the middle left side.
  4. Place the Total Infection Percentage Map in the top right corner.
  5. Place the Highlighted Countries line graph and predictions at bottom.
×

Step 4: Publish the Dashboard to Tableau Public

Save the dashboard and make it visiable to the public.
