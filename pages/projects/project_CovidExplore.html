Covid19 Data Exploration

Infection and death information

Tableau Public Viz

Summary:

This project contains my exploration into Covid19 infections and deaths data. It consists of getting data from the Our World in Data website and processing the downloaded spreadsheet into smaller spreadsheets. I separated the deaths data from the infections data and saved it as a separate Excel file. Then I uploaded the spreadsheets into BigQuery to explore the data with SQL queries. Finally I exported the resulting tables for use in the Covid data visualization project. I followed Alex the Analyst's YouTube video to complete this project. In the future I plan on exploring the data further to determine how different comorbidities affected a patient's outcome.

Results

The data set contained data from January 3, 2020 to August 16, 2023. The dataset contained 67 fields. The dataset was separated into two Excel spreadsheets, one relating to Covid19 deaths and the other to Covid19 vaccinations. The Covid19 deaths Excel spreadsheet contained all columns from the start of the dataset to the weekly_hosp_addmissions_per_million column. The Covid19 vaccinations spreadsheet included the first 4 columns (iso_code, continent,location, date) and all columns after the weekly_hosp_addmissions_per_million column. There was a total of 334,304 line items. Most countries did not start counting Covid19 cases until March 2020 and changed from daily to weekly counts in April 2023.

The columns calculated in this project were:

  1. DeathRate ( percent of infected population that died),
  2. PopulationPercent (percent of the population that got Covid19),
  3. HighestInfectionCount (max number of total cases per location),
  4. PercentPopulationInfected (max PopulationPercent per location),
  5. TotalDeathCount (max total deaths per location),
  6. Total_Cases (sum of all cases),
  7. Total_Deaths (sum of all deaths),
  8. DeathPercentage (death percentage of infected population of the world),
  9. VaccinatedPopulationPercent (percentage of vaccinated population by location).

  1. Go to the source website:ourworldindata.org/covid-deaths.
  2. Scroll to the Covid graph and push the slider all the way to the earliest date.
  3. Download the dataset as a CSV file.

Step 2: Separate the data

Open the dataset in Excel and separate deaths and infection data.

  1. Open the downloaded dataset in MS Excel.
  2. Place the Date column to the left of the Population column.
  3. Remove every column after the weekly_hosp_addmissions_per_million column and save spreadsheet as CovidDeaths.
  4. Remove every column from the weekly_hosp_addmissions_per_million column to population and save spreadsheet as CovidVaccinations.
  5. Create a new dataset in BigQuery
  6. Create the CovidDeaths table and CovidVaccinations table by uploading the spreadsheets with the auto detect schema option selected.
