Movie Industry Correlation

Data Correlation using Python

View code on GitHub

Summary:

I conducted this data analysis project to uncover the key factors influencing movie popularity and to practice using the matplotlib and Seaborn Python libraries. I chose this dataset because it spans movie genres, budgets, and release years. By applying data visualization techniques and statistical analysis, I tested my hypothesis that a movie's budget has a high correlation with it's gross revenue.

Data Source:

Daniel Grijalvas's Kaggle page (September 2023).

Skills & Programs Used:

Step 1: Gather Data

Download the Kaggle movie industry dataset and setup a Jupiter Notebook.

  1. Download the "Movie Industry Three Decades of Movies" dataset from Kaggle (https://kaggle.com/danielgrijalvas/movies).
  2. Launch a new Python 3 notebook in Jupiter Notebooks.
  3. Import the necessary packages and libraries: pandas, seaborn, numpy, matplotlib.
  4. Read the data with a pandas dataframe and view the dataframe header to identify field titles.

Loop through columns and rows with Python.

  2. Drop any rwos that have missing data represented by "NaN".

Step 3: Check data types

Check each column for correct data types.

  1. Check what the data type is for each column with dtypes.
  2. Remove any trailling decimals in the "budget" and "gross" columns by changing the data type to "int64".
  3. Check that the "year" column matches the "released" column's year. Create a new column called "yearCorrect".
  4. Sort the data by the "gross" revenue column.
  5. Drop any duplicate movies from the data frame.

Step 4: Create visuals

Create scatter plots of film budget vs gross revenue and related correlation matrix.

  1. Create a scatter plot of budget vs gross revenue with matplotlib.
  2. Create a scatter plot of budget vs gross revenue with a trendline with seaborn.
  3. View any field correlation values as a table with the Pearson method.
  4. Create a correlation matrix from those values using seaborn and matplotlib.
